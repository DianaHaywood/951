军工项目管理软件安装说明
============================

一、系统要求
-----------
1. 操作系统：Windows 7/8/10/11 (64位)
2. 内存：至少4GB RAM (推荐8GB或以上)
3. 硬盘空间：至少500MB可用空间
4. 显示器分辨率：1280x768或更高

二、安装步骤
-----------
1. 运行安装程序
   - 双击运行 "MilitaryProjectManagement_Setup.exe"

2. 选择安装位置
   - 默认安装路径为：C:\Program Files\军工项目管理
   - 建议不要安装在系统盘，可选择其他硬盘分区

3. 选择组件
   - 主程序（必需）
   - 桌面快捷方式（推荐）
   - 开始菜单快捷方式（推荐）
   - 文件关联（可选）

4. 完成安装
   - 点击"安装"开始安装过程
   - 安装完成后，可选择立即运行程序

三、首次运行配置
---------------
1. 管理员账号设置
   - 首次运行需要设置管理员账号
   - 默认用户名：admin
   - 默认密码：admin123 (首次登录后请立即修改)

2. 数据存储路径设置
   - 建议选择非系统盘存储项目数据
   - 确保有足够的存储空间

3. 安全等级设置
   - 根据项目涉密等级设置相应的安全级别
   - 确保符合国家保密要求

四、使用说明
-----------
1. 项目管理
   - 创建新项目：文件 → 新建项目
   - 导入项目：文件 → 导入
   - 导出项目：文件 → 导出

2. 文件管理
   - 上传文件：点击"上传文件"按钮或拖拽文件到指定区域
   - 文件预览：点击文件后的"预览"按钮
   - 文件操作：支持打开、编辑、删除等操作

3. 进度监控
   - 查看项目进度：在项目概览页面查看进度条
   - 管理进度节点：在"进度计划"选项卡中管理

4. 数据备份
   - 自动备份：系统每5分钟自动保存一次
   - 手动备份：工具 → 数据库维护 → 创建备份
   - 恢复备份：工具 → 数据库维护 → 恢复备份

五、安全注意事项
---------------
1. 保密要求
   - 本系统处理的文件可能涉及国家秘密
   - 使用人员必须经过保密培训
   - 严禁将涉密文件复制到非涉密计算机

2. 数据安全
   - 定期备份重要数据
   - 不要随意删除数据库文件
   - 系统退出时会自动保存当前状态

3. 访问控制
   - 不同用户设置不同的访问权限
   - 记录所有操作日志
   - 定期审计系统使用情况

六、故障排除
-----------
1. 程序无法启动
   - 检查是否已安装.NET Framework 4.7.2或以上版本
   - 尝试以管理员身份运行
   - 重新安装程序

2. 文件上传失败
   - 检查文件是否被其他程序占用
   - 检查磁盘空间是否充足
   - 检查文件权限设置

3. 预览功能异常
   - 确保已安装相关文件预览组件
   - 检查文件格式是否受支持
   - 尝试使用其他程序打开文件

七、技术支持
-----------
如遇问题，请提供以下信息：
1. 操作系统版本
2. 软件版本号
3. 错误提示截图
4. 问题重现步骤

联系方式：
- 内部技术支持电话：XXXX-XXXXXXX
- 保密邮箱：security@example.com

八、卸载程序
-----------
1. 通过控制面板卸载
   - 控制面板 → 程序和功能 → 选择"军工项目管理" → 卸载

2. 使用卸载程序
   - 开始菜单 → 军工项目管理 → 卸载
   - 或运行安装目录下的Uninstall.exe

注意：卸载前请确保已备份所有重要数据！
